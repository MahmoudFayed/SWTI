Cursor symbols

The cursor symbols replace some of the most used ascii ordinals with constants. You can use this list instead of writing a ordinal ascii number. The common keyboard symbols are omitted, they can be used as characters in single quotation marks 'A' or '#'. Some special symbols are not included, for example letters from national alphabet æ and other symbols like ordinal indicator ª. If you don't find your favorite ASCII symbol, add in your code a new integer constant and set it to standard ASCII code number.
